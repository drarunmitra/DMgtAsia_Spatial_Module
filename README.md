# Module 2 — Spatial Epidemiology (consolidated)

Single, self-contained, **runnable** home for Module 2 of *Data Science for Public Health*.
Consolidated 2026-06-26 from the canonical sources identified in
`~/Downloads/Module2_SpatialEpi_content_map.md`. Maps to the `materials/_m2/` layout in
`~/Downloads/Module2_SpatialEpi_build_brief.md`.

## Layout

```
Module2_SpatialEpi/
├── .here                 # sentinel so here() resolves to this root
├── slides/               # canonical lecture decks (.qmd) + styles.css
├── exercises/            # hands-on labs (Ex 1–4, 6, 7, John Snow, SIR/SMR)
├── data/                 # tabular data (csv) + AHS sf
├── spatial_files/        # shapefiles, rasters, prebuilt sf (from spatial_files.zip)
├── capstone/             # CUG reproducible pipeline (Day9) + AHS clustering
├── resources/            # install script, prerequisites, readings
├── assessments/          # SDS4PH_Module1–4 Moodle quiz XML (import-ready)
└── plots/                # exercise output target (empty)
```

`here()` paths in every script resolve against this root — open the folder as an RStudio
project (or rely on the `.here` sentinel) and scripts run in place.

## Run order (Unit C/D chain)

Ex 2 → Ex 3 → Ex 4 share state via `spatial_files/state_med_colleges_sf.rds` (a prebuilt
copy ships, so any exercise also runs standalone):

1. `exercises/hands_on_exercise_1.R` — load India shapefile, CRS 4326↔32643, plot
2. `hands_on_exercise_2.R` — medical-college choropleth, joins → writes `state_med_colleges_sf.rds`
3. `hands_on_exercise_3.R` — raster (`gpw_v4_…tif`) extraction → per-million choropleth
4. `hands_on_exercise_4.R` — geocoding (⚠ hits OSM/Google APIs; Google needs a key)
5. `hands_on_exercise_6.R` / `choropleth_malnutrition_exercise.R` — NFHS-4 vs 5 stunting maps
6. `hands_on_exercise_7.R` — Moran's I (`spdep`) + LISA (`rgeoda`)
7. `john_snow_map_excercise.qmd` — `HistData` (no external file)
8. `SIR_SMR_assignment.Rmd` — Kerala COVID SIR/SMR

Install deps first: `source("resources/install_required_packages.R")`.
